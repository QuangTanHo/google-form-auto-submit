💡 Tự động chạy không cần mở máy chủ (Windows Task Scheduler)

Nếu bạn tắt máy tính hoặc không muốn giữ ứng dụng chạy ngầm, bạn có thể lập lịch trực tiếp bằng công cụ Windows Task Scheduler có sẵn trên hệ điều hành để chạy script này theo đúng giờ hẹn.

1

Mở Task Scheduler trên Windows (tìm trong Start Menu).

2

Chọn Create Basic Task..., đặt tên là Auto Checkin Google Form.

3

Chọn Trigger: Daily, cấu hình các ngày trong tuần và thời gian bắt đầu chạy trùng khớp với giờ Check-in.

4

Action chọn Start a program. Điền các tham số sau:

  • Program/script: node
  • Add arguments: cli.js --type=checkin
  • Start in (Thư mục chứa dự án): C:\Users\quang\.gemini\antigravity\scratch\google-form-auto-submit
5

Tương tự, tạo thêm một Task nữa cho việc Checkout với thời gian tương ứng và argument là cli.js --type=checkout.
